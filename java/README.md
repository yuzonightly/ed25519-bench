# Ed25519 Benchmarks

Ed25519 benchmarks in Java.

Benchmarked libraries:

- Bouncy Castle library.
- `str4d/ed25519-java`.

## Built With

- `JMH`.

## Run Benchmarks

```console
yuzo@earth:~/ed25519-bench/java$ mvn package
yuzo@earth:~/ed25519-bench/java$ java -jar target/ed25519-benchmarks-0.1.0.jar
```
